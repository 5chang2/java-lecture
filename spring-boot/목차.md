## 강의 진행 방법

### 권장 학습 순서
1. **Part 1-2**: Spring Boot 기초와 데이터베이스 연동을 먼저 학습
2. **Part 3**: Instagram 프로젝트를 단계적으로 구현 (챕터 6 → 7 → 8)
3. **Part 4**: 핵심 기능을 프로젝트에 적용
4. **Part 5**: 테스트 작성 및 배포

### 실습 중심 학습
- 각 섹션의 예제 코드를 직접 작성
- Instagram 프로젝트를 단계별로 완성
- `starter` 코드로 시작 → `solution` 코드로 검증

### 학습 시간
- **전체 과정**: 약 50-60시간
- **기초 (Part 1-2)**: 15-20시간
- **프로젝트 (Part 3)**: 15-20시간
- **고급 (Part 4-5)**: 15-20시간
